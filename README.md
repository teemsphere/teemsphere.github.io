# teems-manual

Source for the [teems user manual](https://teemsphere.github.io/), built with [Quarto](https://quarto.org/), inspired by the [targets user manual](https://github.com/ropensci-books/targets/).

## Rendering

```bash
quarto render
```

Output is written to `_book/`.

## Structure

Each chapter is a `.qmd` file. Chapter order is defined in `_quarto.yml`.

## Contributing

To edit a page, click the **Edit this page** link in the rendered book — it opens the source `.qmd` directly on GitHub.
